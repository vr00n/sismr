# Staten Island Society of Model Railroaders, Inc.

**Building Community, One Track at a Time — Since 1956**

This is the official website for the Staten Island Society of Model Railroaders, Inc. (SISMR), one of Staten Island's longest-running hobby clubs.

## Live Site

Visit the site at: **https://vr00n.github.io/sismr/**

## About the Club

Founded in 1956, the SISMR brings together model railroad enthusiasts of all skill levels. Members build detailed HO-scale (1:87) layouts, participate in community shows, and share the joy of the hobby with Staten Island.

## Adding Photos

To add your own photos to the gallery:

1. Add image files to the `images/` folder (JPEG or PNG recommended)
2. In `index.html`, find the `gallery-grid` section
3. Replace a `gallery-placeholder` div with an `<img>` tag:

```html
<div class="gallery-item">
  <img src="images/your-photo.jpg" alt="Description of photo">
</div>
```

## Deployment (GitHub Pages)

This site is designed to be hosted on GitHub Pages:

1. Push this repository to GitHub
2. Go to **Settings > Pages**
3. Set source to **Deploy from a branch** and select **main** / **(root)**
4. The site will be live at `https://<username>.github.io/<repo-name>/`
